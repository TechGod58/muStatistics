                                                          Estimate internal consistency across a selected scale using Cronbach alpha.

                                                          Factor analysis

                                                          Run a first-pass exploratory factor extraction on a selected numeric field set.

                                                          Exploratory extraction supports principal components or principal axis with optional varimax/promax rotation.

                                                          Categories / optimal scaling

                                                          Quantify categorical levels over a one-dimension latent scale with optional anchor guidance.

                                                          Results open in the Output Viewer.

                                                          Conjoint analysis

                                                          Estimate attribute part-worth utilities and profile predictions from preference ratings.

                                                          Results open in the Output Viewer.

                                                          Direct marketing

                                                          Build RFM/response propensity, segment profiles, and lift-style deciles for targeting.

                                                          At least one of response, recency, frequency, or monetary is required.

                                                          Forecasting

                                                          Run a first-pass linear trend forecast for ordered or date-like data.

                                                          Results open in the Output Viewer.

                                                          Cluster analysis

                                                          Run deterministic k-means clustering over standardized numeric fields.

                                                          Results open in the Output Viewer.

                                                          Decision tree

                                                          Run CART or random forest classification with configurable splitting controls.

                                                          Results open in the Output Viewer.

                                                          GLM / ANCOVA

                                                          Fit a numeric dependent variable with categorical factors and numeric covariates.

                                                          Factors are dummy-coded with a baseline level. Results open in the Output Viewer.

                                                          Mixed model (random intercept)

                                                          Fit fixed predictors with group-level random intercept adjustments.

                                                          Current implementation supports random intercepts. Results open in the Output Viewer.

                                                          Generalized estimating equations (GEE)

                                                          Run cluster-aware gaussian or binomial models with robust covariance.

                                                          Binomial family expects a binary dependent outcome. Results open in the Output Viewer.

                                                          Repeated measures

                                                          Compare paired numeric measurements across time points or conditions.

                                                          Complete rows across the selected measures are included. Results open in the Output Viewer.

                                                          Survival analysis

                                                          Build Kaplan-Meier survival tables from time and event status fields.

                                                          Event fields accept values like 1/0, true/false, yes/no, event/censored. Results open in the Output Viewer.

                                                          Complex samples

                                                          Create survey-design aware means or proportions with optional strata, cluster, and domain fields.

                                                          Use the Quantitative analysis controls above for weight field selection. Results open in the Output Viewer.

                                                          Neural network

                                                          Run a deterministic single-hidden-layer model for exploratory prediction.

                                                          This is exploratory apparent-fit output; train/test validation is a later expansion. Results open in the Output Viewer.
